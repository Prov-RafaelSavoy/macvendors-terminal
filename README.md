# macvendors-terminal

## Use

Get vendor of MAC without directly from terminal, without need access webpage.

```
$ macvendor b8:af:67:21:33:f1
Hewlett Packard
```



## Installation

### Requirements

Curl: https://curl.se/

1 - Download the file

Gitclone 

2 - Move file to /etc/bin/

sudo mv macvendors /etc/bin/macvendors

3 - Use!

macvendors
